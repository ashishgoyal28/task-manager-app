# Task Manager App

A full-stack MERN Task Manager application with JWT authentication and CRUD functionality.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Create Tasks
- View Tasks
- Update Task Status
- Delete Tasks
- Logout Functionality

## Tech Stack

### Frontend
- React
- Axios
- React Router DOM
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Installation

### Clone Repository

```bash
git clone https://github.com/ashishgoyal28/task-manager-app
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=3001
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
```
